# Melbourne E-Bike Usage Analysis: Demand Drivers & Business Insights

## Project Overview

This project analyzes Melbourne’s e-bike sharing data to identify key demand drivers, user behavior patterns, and operational insights.
The analysis explores temporal trends, user segmentation, weather impact, and station-level performance to translate data into actionable business recommendations.

## Objectives

Identify peak hourly and daily demand patterns
Compare weekday vs weekend usage behavior
Analyze seasonal demand fluctuations
Evaluate subscriber vs casual rider behavior
Assess the impact of temperature and rainfall on ridership
Determine high-traffic station locations

## Key Findings

Ridership peaks during late night (1–3 AM), midday (11 AM–1 PM), and evening (5–8 PM), indicating both commuter and leisure usage.
Weekend late-night demand is significantly higher than weekdays, suggesting nightlife-driven activity.
Demand is strongly seasonal, with peaks in February–March and August–October.
Subscribers generate the majority of trips, though trip characteristics are consistent across user types.
Ridership is highest in moderate temperatures (15–25°C) and declines sharply in extreme weather conditions.
The network operates as a multi-node system, with several high-traffic stations rather than one dominant hub.
